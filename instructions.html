SETUP WORDPRESS ON DOCKER (ubuntu)

Pull the following images from docker hub

Download only the official image or the one with the highest stars
  • Mysql ( docker pull mysql )
  • Phpmyadmin ( docker pull phpmyadmin/phpmyadmin )
  • Wordpress ( docker pull wordpress )

This setup uses docker-compose (much easier setup)

  1. Step 1. Install Docker for Ubuntu / Windows / Mac
  2. Step 2. Pull images from docker hub.
  3. Step 3. Copy docker-compose.yml
How to get inside docker container - docker exec -it /bin/bash (to get a bash shell in the container) - Generically, use docker exec -it
